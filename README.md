## 🌟 主要功能

- **定时提醒**：自定义工作时长，时间一到，蟑螂就会从屏幕四周爬出。
- **自定义召唤**：最高支持同时召唤 **50 只** 蟑螂，视觉冲击力拉满。
- **灵活配置**：
  - 调整蟑螂的**数量**、**大小**和**移动速度**。
  - 设置快速蟑螂出现的概率。
  - 自由设定休息时长。
- **智能循环**：休息结束后自动重新开始工作计时，无需手动干预。
- **全平台支持**：提供 Windows (.exe) 和 macOS (.dmg) 安装包。
- **汉化支持**：完整的中文界面与系统通知。

## 🚀 快速开始

### 安装
- **Windows**: 下载并运行 `蟑螂提醒 Setup 1.0.0.exe`。
- **macOS**: 下载 `蟑螂提醒-1.0.0-arm64.dmg`，将应用拖入 `Applications` 文件夹。

### 开发与运行
1. 安装依赖：
   ```bash
   npm install
   ```
2. 本地开发预览：
   ```bash
   npm start
   ```
3. 打包 Windows 版本：
   ```bash
   npm run build:win
   ```
4. 打包 macOS 版本：
   ```bash
   npm run build:mac
   ```

## ⚙️ 设置说明

右键点击系统托盘（任务栏）中的🪳图标，选择“设置”即可进入配置页面：
- **休息间隔**：设置两次休息之间的时间。
- **显示时长**：蟑螂在屏幕上停留的时间。
- **蟑螂数量**：设置召唤的规模（1 - 50只）。
